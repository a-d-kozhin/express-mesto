# Проектная работа 12

## Технологии
Express JS
Eslint (Airbnb base)

## Функционал
API с 3 роутами:
1) GET http://localhost:3000/users; Ответ: JSON с пользователями;
2) GET http://localhost:3000/cards; Ответ: JSON с карточками;
3) GET http://localhost:3000/users/:id; Ответ: JSON пользователя с соответствующим id;
